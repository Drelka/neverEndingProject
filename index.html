Creating variables

Const creates a variable whose reference cannot be reassigned, while let creates a variable that can be reassigned.



const name = "Joe" //This can never change
let age = 22

name = "Jamie" //throwing error
age = 33 //new value is valid

Ternary operator

...is a one-line code for a simple "if...else" statement and returns one of two expressions, just such:



condition ? expression_if_true : expression_if_false;


let retirementAge = 65;
let myAge = 35;
let isRetired = myAge >= retirementAge ? "Retired" : "Not retired";
console.log(isRetired);

Backquotes / Backticks

... are used to define template literals - a special type of string allowing us to embed expressions and JavaScript in them. You can also declare them over multiple lines.



let name = "Christina";
let greeting = `Hello, ${name}. Nice to meet you.`;

let age = 22;
let answer = `Hi. My name is ${name}. I'm ${age} years old.`;

let brotherAge = 15;
console.log(`I am ${age - brotherAge} years older than my brother.`);
